# CoParent Talk

A premium, secure family-law-compliant communication and scheduling hub designed for co-parents to manage joint custody, shared schedules, split expenses, and exchange logs.

## Features

- 📅 **Joint Custody Calendar**: Track parenting schedules, request custody swaps, and manage child-related events.
- 💬 **Secure Message Board**: Keep communications transparent and objective. Includes a built-in sentiment analysis tool to flag and encourage constructive tone.
- 💸 **Shared Expense Tracker**: Log joint child-rearing expenses, document receipts, and settle balances transparently.
- 📍 **GPS Swap Check-ins**: Log parenting exchanges with GPS-verified coordinates for legal compliance and peace of mind.
- ⚖️ **Court Admissible Exports**: Export complete communication logs, calendar histories, expense worksheets, and check-in logs as formatted reports.
- 📬 **Simulated Email Notifications**: Real-time simulated outbox notifications when co-parents add events, messages, expenses, or log check-ins.

## Tech Stack

- **Core**: React, TypeScript, Vite
- **Icons**: Lucide React
- **Styling**: Premium Vanilla CSS (Responsive Layout, Glassmorphic elements)

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server locally:
   ```bash
   npm run dev
   ```
4. Build the static distribution bundle:
   ```bash
   npm run build
   ```
